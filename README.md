# LeetCode Solutions in C++
A curated collection of LeetCode problem solutions with comprehensive explanations, complexity analysis, and test coverage.
## Project Goals
- **Educational**: Clear explanations of algorithms and data structures
- **Production-quality**: Modern C++20, comprehensive tests, clean code
- **Discoverable**: Organized by problem name with searchable index
## Quick Start
### Prerequisites
- CMake 3.14+
- C++20 compatible compiler (GCC 10+, Clang 10+, MSVC 2019+)
### Build and Run
```bash
# Clone the repository
git clone <your-repo-url>
cd leetcode
# Build
cmake -B build -S .
cmake --build build
# Run main demo
./build/leetcode
# Run tests
cd build && ctest --output-on-failure
```
